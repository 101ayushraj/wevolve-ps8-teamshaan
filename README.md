## ???? Project Overview
Wevolve Gen is an all-in-one platform that aims to close the gap that exists between employers and job applicants.

1. For Employers: It employs Generative AI (Google Gemini) to craft, in milliseconds, unbiased, search-engine-optimized job descriptions.

2. For Candidates: It offers an intelligent “Skills Gap Analysis” engine that compares a candidate’s existing profile against industry-targeted goals in order to develop a learning strategy.
